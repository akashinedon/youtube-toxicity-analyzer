# 🎥 YouTube Toxicity Analyzer

AI-powered tool to analyze YouTube video comments for toxicity and sentiment.

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

## ✨ Features

- 🤖 **AI-Powered Toxicity Detection** - 7 toxicity categories using Detoxify (BERT-based)
- 😊 **Sentiment Analysis** - Positive, Negative, Neutral classification
- 📊 **Interactive Visualizations** - 10+ chart types with Plotly
- ⏰ **Timeline Analysis** - Map toxic comments to video timestamps
- 🆚 **Video Comparison** - Compare up to 3 videos side-by-side
- 📥 **Export Reports** - HTML, CSV, and JSON formats
- 📜 **Analysis History** - Track your recent analyses

## 🛠️ Technologies

- **Python 3.8+**
- **Streamlit** - Web framework
- **Detoxify** - Toxicity detection AI
- **Plotly** - Interactive charts
- **Pandas** - Data processing
- **YouTube Data API v3** - Comment fetching
